Introduction

With the spread of the new coronavirus, it has had a huge impact on the world. In the process of studying the new coronavirus, the processing and analysis of data is extremely important. Predicting data within a certain period of time can help people better respond to the epidemic, so as to achieve maximum utilization of resources and faster response to emergencies. In this project, the British new crown virus data is predicted and analyzed by using machine learning methods. Predicting from the four characteristics of the number of daily tests in the UK, the number of new cases per day, the number of patients in the hospital, and the number of vaccinated patients, the number of deaths of people with positive test results within 28 days can be obtained. The predicted number of deaths can help the government more intuitively estimate the situation of the epidemic, and it can also allow hospitals to better face the upcoming pressure.

In this project, machine learning is mainly applied. Machine learning is a science related to artificial intelligence. The main purpose is to allow computers to improve specific algorithms in empirical learning, thereby optimizing computer programs. The three elements of machine learning are data, algorithms, and models. Machine learning is to analyze and process data by selecting appropriate algorithms, and make predictions based on the results of the processing and new data. The obtained data set needs to undergo related processing, and the data can be preprocessed by reshape, changing the data type, and other methods.

In deep learning, neural network is the most widely used network system. In this project, the prediction of the death toll was achieved by using a neural network model. The neural network is mainly composed of three parts, the input layer, the output layer, and the hidden layer. In this project, the input layer is the number of daily tests in the UK, the number of new cases per day, the number of patients in the hospital, and the number of people vaccinated, and the output layer is the number of deaths of people with positive test results within 28 days. By comparing the results of the output layer and the test set, the accuracy rate corresponding to the number of days can be obtained. So as to realize the prediction and verification of the data.

Machine learning framework and code

In this project, the network structure of machine learning is a neural network model. As shown in Figure 3 below. It is mainly an algorithmic mathematical model for distributed parallel information processing. After the input layer receives the information transfer, it will be passed to the output layer, and each neuron will have a weight.

Figure 5: Neural Network

In this architecture, because the amount of data is not large, the batch size could be set to 1. batch size refers to the number of samples lost in one training, and its size will affect the optimization and speed of the model. Before batch size is used, when the network is trained, all the data will be input into the network at once, and then their gradients will be calculated for back propagation. In a database with a small number of samples, it is feasible not to use batch Size, and the effect is also good. If the batch size is increased, in order to achieve the same accuracy, the epoch must be increased. Therefore, judging from the database of this project, the epoch has not changed much.

Secondly, in the input shape of the neural network structure, its shape has also changed from 8 to 4, mainly due to the change in the number of features. In this neural network structure, the used activation function is relu function. It is a non-linear function represented by the ramp function and its variants. The relu function can achieve unilateral suppression, which means that when training a model, there are often not many features related to the target. Therefore, the sparse model can be achieved through relu function, and it would be better for digging out relevant features and fitting training data.

Finally, in the output layer, only one neuron is used and its activation function is a linear activation function. In the process of calculating loss, the MSE function is used to calculate the loss of the training set and the test set. As shown in Figure 6.

Conclusion

In this project, machine learning is used to predict the number of deaths of positive testers within 28 days from four aspects: the number of patients per day, the number of patients in the hospital, the number of vaccinations, and the number of people tested. In the data set, the feature value has a certain correlation, so the target value prediction result is not low. However, in the forecasting process, because the number of deaths occasionally is 0 for a period of time, and the predicted number is not 0, in this case, the accuracy rate will be 0. In addition, the number of vaccinated persons is 0 at the beginning of the epidemic, so this characteristic value will also cause errors for a period of time. Through the analysis of the prediction results of the model within 15 days, the model still needs to be improved. Choosing a more suitable time interval can improve the accuracy of the model.

In addition, the data set was also preprocessed in this project. The data set needs to be selected with the appropriate data type to achieve the purpose of inputting into the model. For example, convert from object to float64. Moreover, in this project, the data set is divided into a test set and a training set, and the data set is standardized to make the changed data more suitable for learning algorithms.

Finally, in the training model, because the sample size is not large, batch size is set to 1. In this project, it mainly uses a neural network structure, the activation function in the model is the Relu function, and the activation function in the output layer is a linear activation function. The Relu function is actually a piecewise linear function, which can turn all negative values ​​into 0, while the positive values ​​remain unchanged. There are several advantages for Relu activation function. Compared with linear functions, Relu has a stronger expressive ability. For nonlinear functions, Relu does not have the problem of gradient disappearance because the gradient of the non-negative interval is constant, and the convergence rate of the model can be maintained in a stable state. The disappearance of the gradient means that the error between the predicted value and the true value will be attenuated every time it propagates through one layer, which will cause the model to converge.
